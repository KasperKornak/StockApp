DivTax docs

DivTax is a platform meant to automatically calculate dividend tax received from US-registered companies (or any company which has ADR/ADS available).

Positions page

The next step is to add your positions. Hit the Positions button at the top-right corner of the page. you will be redirected to the following page:

This page allows you to add, edit or delete existing positions, as well as displays some basic info about your holdings. You might have seen the DELETED_SUM position is already in your portfolio. Don't worry - this position stores data about dividends which you received, but you no longer have a position in the company. You can finf more about that below.

Adding a position

Now, let's create first position in portfolio. We'd like to add 50 shares of Apple. To do that, simply hit the Add Position button, then the following form will appear:

Now let's go over the fields required to be filled.

  1. Ticker

    2. Pretty straightforward - the ticker of the company you would like to add your portfolio. In our case it will be AAPL.

  2. Shares

    3. Number of shares we would like to add. In our case it'll be 50.

  3. Domestic Tax

    4. Number in percentages of the tax rate for the position in Poland. For instance, if you have filled in W-8BEN form and privilages given by this form were granted to you, in 95% of situations your tax rate will be 4 per cent. However, if your broker doesn't allow you to fill W-8BEN or effective tax rate is different you can obviously fill in Domestic Tax field however you like. In our case, the Domestic Tax field will be filled by 4 value representing 4% tax to be paid in Poland.

  4. Div YTD

    5. Dividends year to date received from given company. Let's say this year so far, Apple paid us 19.34USD.

  5. Div PLN

    6. Dividend tax in PLN to be paid this year. For instance, let's say Apple paid us dividends in two tranches:

    1. First quarter - 10.03.2023 - 8.34USD

      2. So the dividend tax to be paid for the first quarter is: 8.34 * 0.04 * 4.4356 = 1.48PLN, where the last element from the formula is an exchange rate from NBP tables.

    2. Second quarter - 12.07.2023 - 11.00USD

      3. So the dividend tax to be paid for the second quarter is: 11.00 * 0.04 * 4.0374 = 1.78PLN, where the last element from the formula is an exchange rate from NBP tables.

    As you can see, calculating this by hand may be quite exhausting, so I recommend using some external services like podatekgieldowy.pl to handle this calculations for you.

Editing position

The process of editing your position is basically the same as adding position. The same form appears when Edit Position button is clicked. All you have to do is fill in the fields that you wish to be changed before hitting the Edit button.

Deleting position

Deleting your position is quite easy. At the Positions webpage, hit the Delete Position button, then enter the ticker of the position you wish to be deleted:

After hitting the Delete button you will see that the dividends from the deleted position were transfered to DELETED_SUM position.

API limits

Due to the current API plan limits, the platform is restricted to 5 API calls/minute. Because of that, sometimes adding a stock to portfolio may not work if many users are simultaneously updating the portfolio. If this happens to you, please wait about minute and try to add it again. If the problem doesn't stop - contact me.

There is a built-in mechanism which restricts the backend from sending unnecessary calls to API - if some user adds a position to their portfolio, which somebody else has added before, the call won't be made. However, adding a position which doesn't already have an entry in database requres making the call. As the user base grows, I expect that the problem will occur less and upgarding data provider plan would definitely help.

Different regions

Due to some bad decisions during early development of the website, backend and database are located on different continents, which visibly affects the speed of the website. In the near future I plan on migrating the whole infrastructure to Europe.
